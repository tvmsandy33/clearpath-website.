Investment Driver Application

The application enables administrators to manage investment pricing dynamically, with a focus on region-specific pricing.

Admins can set and update price tiers for each investment based on factors like local market conditions, taxes, or currencies, using a centralized control panel. The system detects a user's region through their account or IP address and displays the appropriate pricing.

It supports complex scenarios, including tax rates, special offers, and discounts tailored to regions. Admins can track historical pricing changes and implement personalized pricing strategies. Users are notified of pricing updates, ensuring transparency and consistency. This flexible, region-based pricing system enhances user experience and enables fair, localized investment pricing for both users and administrators.

Centralized File System

The Centralized File System, developed for a Telecom client, automates document organization within a hierarchical folder structure.

Users can define parent and child folders via a user-friendly interface, with the system validating the structure against business rules to prevent errors like duplicate names.

Appian integration dynamically creates the folder structure in Google Drive, while robust error-handling ensures reliability by addressing issues such as insufficient permissions. Automated email notifications keep users informed when folders and documentation are ready. Role-based access controls protect sensitive information, allowing only authorized users to manage specific folders, ensuring secure and efficient document management.
